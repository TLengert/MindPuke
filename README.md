# MindPuke

MindPuke is a high-performance, node-based mind mapping canvas built with React, Vite, and React Flow. It features a premium, dark-mode-first design designed for AI-driven brainstorming.

## Tech Stack

- **Framework:** React + Vite (TypeScript)
- **Canvas:** `@xyflow/react` (React Flow)
- **State Management:** `zustand`
- **Styling:** `tailwindcss` (v4)
- **Icons:** `lucide-react`

## Features

- **Full-Screen Canvas:** Optimized for maximum focus.
- **Dark Aesthetic:** Pure black background (#000000) with a subtle dot grid.
- **Snap-to-Grid:** Ensures alignment and order at all times.
- **Scalable Architecture:** Built with Zustand for complex data structures and future AI integrations.

## Getting Started

1.  **Install dependencies:**
    ```bash
    pnpm install
    ```

2.  **Start development server:**
    ```bash
    pnpm dev
    ```

3.  **Build for production:**
    ```bash
    pnpm build
    ```
